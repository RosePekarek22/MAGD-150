# MAGD-150
GitHub Project Assignment 


### Lab 3
Meteors and Comets flying across a night sky using **pmouse/mousex** and **pmouse/mouseY** to create the Meteor tail and _frameCount % width_ to move the Comets

### Lab 5
A brown owl with hypnotizing, color changing eyes made using _while loops_. Its mouth opens and closes using _mousePressed_ and _mouseReleased_, and it can blush using the **D** and **F** keys using _keyPressed_.

### Lab 6
A simplified Frogger. Logs move across the water using _arrays_ and _for loops_. The frog moves using _arrays_, _for loops_, and **mousex/mousey**.
